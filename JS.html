What is JavaScript?

JavaScript is a powerful programming language that adds interactivity and dynamic behavior to websites. While HTML provides structure and CSS handles styling, JavaScript brings your web pages to life.

JavaScript can update content dynamically, respond to user actions, validate forms, create animations, and communicate with servers without reloading the page.

How to Add JavaScript

There are three ways to add JavaScript to your HTML:

1. Inline JavaScript

<button onclick="alert('Hello!')">Click Me</button>

2. Internal JavaScript

<script>
    function greet() {
        alert('Hello World!');
    }
</script>

3. External JavaScript (Recommended)

<script src="script.js"></script>

JavaScript Basics

Variables

Variables store data values:

let name = "John";
const age = 25;
var city = "New York";

Data Types

let text = "Hello"; // String
let number = 42; // Number
let isTrue = true; // Boolean
let fruits = ["apple"]; // Array
let person = {name: "Bob"}; // Object

Functions

function add(a, b) {
    return a + b;
}

let result = add(5, 3); // result is 8

Common JavaScript Operations

Operation Description Example
Select Element Get HTML element document.getElementById("myId")
Change Content Modify HTML content element.innerHTML = "New text"
Change Style Modify CSS properties element.style.color = "red"
Add Event Respond to user actions button.addEventListener("click", fn)
Show Alert Display popup message alert("Hello!")
Console Log Print to console console.log("Debug info")

Control Structures

Conditional Statements

if (age >= 18) {
    console.log("Adult");
} else {
    console.log("Minor");
}

Loops

// For loop
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// While loop
while (count < 10) {
    count++;
}

DOM Manipulation

The Document Object Model (DOM) allows JavaScript to interact with HTML elements:

Method Description Example
getElementById() Select by ID document.getElementById("demo")
querySelector() Select by CSS selector document.querySelector(".class")
createElement() Create new element document.createElement("div")
appendChild() Add child element parent.appendChild(child)
remove() Remove element element.remove()

Events

JavaScript can respond to user interactions:

  • click: When an element is clicked
  • mouseover: When mouse hovers over element
  • keypress: When a key is pressed
  • submit: When a form is submitted
  • load: When page finishes loading
button.addEventListener('click', function() {
    alert('Button clicked!');
});

Why Learn JavaScript?

  • Interactivity: Create dynamic, responsive user interfaces
  • Versatility: Use for frontend, backend (Node.js), mobile apps, and more
  • In-Demand Skill: One of the most popular programming languages
  • Large Ecosystem: Thousands of libraries and frameworks available
  • Browser Native: Runs in every web browser without installation
  • Community: Massive community with extensive resources and support

Get Started

Ready to make your websites interactive? Open your browser's developer console (F12) and start experimenting with JavaScript today. Try creating buttons that respond to clicks, forms that validate input, or animations that bring your page to life!
