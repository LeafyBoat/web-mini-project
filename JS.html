DOM Patterns

Select, update, and animate

  • Use querySelector for flexible targeting.
  • Keep updates small and predictable.
  • Add or remove classes to trigger CSS states.
const card = document.querySelector(".card");
card.classList.add("is-active");
card.textContent = "Updated content";

Events

Turn user intent into action

const button = document.querySelector(".btn");
button.addEventListener("click", () => {
  button.textContent = "Saved!";
});

Async

Handle async work with clarity

Async flows keep apps responsive. Use async/await to read like synchronous code.

async function getData() {
  const response = await fetch("/data.json");
  const data = await response.json();
  return data;
}

Debugging

Make the console your co-pilot

  • Use console.table for arrays and objects.
  • Group related logs with console.group.
  • Handle errors with try/catch and clear messages.

JavaScript pro checklist

  • Event listeners are scoped and cleaned up when needed.
  • DOM updates happen with class toggles, not inline style.
  • Async calls handle errors gracefully.
  • Console logs are removed before shipping.
