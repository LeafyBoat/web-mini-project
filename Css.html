What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of HTML documents. While HTML provides the structure and content, CSS controls how that content looks.

CSS allows you to control colors, fonts, layouts, spacing, animations, and much more, transforming plain HTML into visually appealing web pages.

How to Add CSS

There are three ways to add CSS to your HTML:

1. Inline CSS

<p style="color: blue; font-size: 18px;">This is blue text</p>

2. Internal CSS

<head>
    <style>
        p {
            color: blue;
            font-size: 18px;
        }
    </style>
</head>

3. External CSS (Recommended)

<link rel="stylesheet" href="styles.css">

CSS Syntax

CSS rules consist of a selector and a declaration block:

selector {
    property: value;
    property: value;
}
Example: h1 { color: red; font-size: 24px; }

Common CSS Properties

Property Description Example
color Text color color: #333;
background-color Background color background-color: #fff;
font-size Size of text font-size: 16px;
margin Outer spacing margin: 10px;
padding Inner spacing padding: 20px;
border Element border border: 1px solid black;
display Display type display: flex;
width / height Element dimensions width: 100%;

CSS Selectors

Selector Type Syntax Description
Element p { } Selects all <p> elements
Class .className { } Selects elements with class="className"
ID #idName { } Selects element with id="idName"
Universal * { } Selects all elements
Descendant div p { } Selects all <p> inside <div>
Pseudo-class a:hover { } Selects links on hover

The Box Model

Every HTML element is essentially a box consisting of:

  • Content: The actual content (text, images, etc.)
  • Padding: Space between content and border
  • Border: A line around the padding
  • Margin: Space outside the border
div {
    width: 300px;
    padding: 20px;
    border: 2px solid black;
    margin: 10px;
}

Why Learn CSS?

  • Design Control: Create beautiful, professional-looking websites
  • Responsive Design: Make websites work on all devices and screen sizes
  • User Experience: Improve readability and navigation
  • Separation of Concerns: Keep structure (HTML) separate from presentation (CSS)
  • Reusability: One CSS file can style multiple HTML pages
  • Career Essential: CSS is a fundamental skill for web developers and designers

Get Started

Ready to style your web pages? Start by creating a CSS file and linking it to your HTML. Experiment with different properties and see how they change your page's appearance!
